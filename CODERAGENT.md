1. **Dynamic Header Detection:** Use `df.count(axis=1).idxmax()` on the first 30 rows to find the actual header row.
2. **Missing Data:** Convert string symbols like `'-'` or `'..'` to `np.nan` using `na_values=['-', '..']`.
3. **Date Parsing:** Many date columns are formatted as `MMM-YYYY` (e.g., `Jun-2025`) or full datetime objects. Check the first few rows of each DataFrame to determine the format.
4. **Sheet Selection:** In all cases, the primary data is located in the first sheet (`xl.sheet_names[0]`).
